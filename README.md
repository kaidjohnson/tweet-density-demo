# Tweet Density Demo #

## Getting Started ##

1. `git clone <repo>`
2. Edit config.json using your Twitter API credentials, which can be created [here](https://apps.twitter.com/)
3. `npm install`
4. `npm run socket`
5. Navigate to [codepen demo](http://codepen.io/kaidjohnson/pen/ORkBvz?editors=0010)
